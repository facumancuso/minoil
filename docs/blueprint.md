# **App Name**: Minoil Central

## Core Features:

- KPI Dashboard: Display key performance indicators such as orders in process, completed, pending, and urgent, with at-a-glance summaries.
- Interactive Visualizations: Generate interactive charts, including a donut chart for component type distribution and a line graph for order trends over the last 6 months. A generative AI tool should suggest relevant charts based on patterns.
- Detailed Order Table: Present a sortable, filterable table of recent orders with detailed information including ID, Client, Component, Equipment, Status, and Progress. Supports filtering and sorting.
- Visual Progress Indicators: Integrate progress bars within the order table to visually represent the completion status of each order, using distinct color-coded statuses.
- Service Insights: Showcase panels displaying most requested services with quantities and the average service time per component. Also includes current status and technical capabilities of the test bench.
- Process Management: Comprehensive system for managing work order processes with defined stages, allowing users to upload new orders and update their statuses with detailed metrics.
- Order Upload and Status Update: Enable users to upload new work orders and update the status of existing orders, capturing detailed metrics at each stage. This will automatically recalculate trends in the AI-enabled visualizations tool.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37), representing the mining industry.
- Background color: Light beige (#F5F5DC), a desaturated version of the primary to provide contrast.
- Accent color: Dark orange (#FF8C00), to highlight important actions and information.
- Body font: 'PT Sans', a humanist sans-serif font for clear and readable text.
- Headline font: 'Playfair', a modern serif font for titles and headers.
- Use industry-specific icons related to mining equipment and operations, with a consistent, modern style.
- Responsive layout to ensure optimal viewing experience on different screen sizes.
- Subtle animations to add dynamism to the dashboard.
